# RESTful Booker API Testing

## Project Overview

This project demonstrates API testing using Postman on the RESTful Booker application. It covers CRUD operations, authentication, response validation, and negative testing.

## Tools Used

- Postman
- Newman
- REST API
- JSON
- GitHub

## APIs Tested

- Health Check
- Authentication
- Create Booking
- Get Booking
- Update Booking
- Partial Update
- Delete Booking

## Validation Performed

- Status Code Validation
- Response Time Validation
- JSON Body Validation
- Environment Variables
- Dynamic Booking ID
- Authentication Token
- Positive Test Cases
- Negative Test Cases

- Postman Collections

- Bug Reports
